# face-sight-test
視線表示をさせたかったので簡単なモック
https://takex5g.github.io/face-sight-test/

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
